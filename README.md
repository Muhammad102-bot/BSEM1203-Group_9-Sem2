--------------------------------------------------------
🚀 OVERVIEW
--------------------------------------------------------
CHERS is a modern desktop-based Clinical Management System
built with Python. It provides a lightweight hospital OS
interface for managing patients, appointments, chat logs,
and emergency responses in real time.

Designed for:
- Hospitals
- Clinics
- Medical training systems
- Student tech exhibitions

--------------------------------------------------------
🔥 FEATURES
--------------------------------------------------------

✔ Secure Login System  
✔ Role-Based Access (Admin / Doctor / Nurse)  
✔ Patient Records Management  
✔ Live Search Functionality  
✔ Appointment Booking System  
✔ Emergency Alert Trigger System  
✔ Audit Log Tracking  
✔ Chat Communication Panel  
✔ Interactive Dashboard UI  
✔ Modern Dark-Themed Interface  

--------------------------------------------------------
🧠 CORE TECHNOLOGIES
--------------------------------------------------------
- Python 3.x
- CustomTkinter (UI Framework)
- Tkinter (Base GUI Engine)
- JSON (Data Storage Support)
- Time Module (Logging & Events)

--------------------------------------------------------
📊 DASHBOARD MODULE
--------------------------------------------------------
- Live patient statistics
- Status tracking (Active / Critical / Pending)
- Visual metric cards
- Real-time system updates

--------------------------------------------------------
👥 PATIENT SYSTEM
--------------------------------------------------------
Each patient record includes:
- Name
- Age
- ID
- Phone Number
- Medical Status

Features:
- Live search filtering
- Status tagging system
- Expandable record structure

--------------------------------------------------------
📅 APPOINTMENT SYSTEM
--------------------------------------------------------
- Book appointments with:
  • Patient Name
  • Symptoms
  • Time slot
- Stored in session memory / database-ready structure

--------------------------------------------------------
🚨 EMERGENCY SYSTEM
--------------------------------------------------------
- One-click emergency trigger
- Instant alert notification
- Logs event into audit system
- Broadcast-ready structure for expansion

--------------------------------------------------------
🔐 SECURITY SYSTEM
--------------------------------------------------------
- Login authentication layer
- Role-based UI access control
- Future-ready password encryption module support
